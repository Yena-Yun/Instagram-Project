## 🎀 구현화면
로그인페이지와 메인페이지를 <b>반응형</b>으로 구현하였습니다.

## 🌊 로그인페이지 (구현기능 5가지)
1. setInterval로 휴대폰 이미지 무한 로테이션
2. input창에 커서를 두면 placeholder 텍스트가 위로 올라감
3. 올바른 아이디와 비밀번호 입력 시에만 로그인 버튼 활성화
4. 비밀번호 숨기기 또는 표시 기능
5. 로그인 성공 시 메인 페이지로 이동하기
